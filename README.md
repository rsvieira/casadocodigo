# casadocodigo
Projeto de estudo baseado nos cursos "Spring MVC I: Criando aplicações web" e "Spring MVC II: Integração, cache, segurança e templates" do Alura.

Usando as seguintes dependências:

* Spring MVC
* Spring Security
* Hibernate
* Guava (cache)
* JUnit
* MySQL/Postgre
* Jackson (JSON)
* Javax (Mail)
